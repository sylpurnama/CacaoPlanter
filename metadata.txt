about=CacaoPlanter helps agronomy and GIS teams plan cacao planting layouts directly in QGIS.

  Key features:
  - Generate planting points (triangular and square grid patterns)
  - Shade tree planning with configurable ratio and buffer
  - Slope analysis from elevation rasters (DTM, DSM, SRTM, LiDAR, drone DEM)
  - Global CRS selector for projected coordinate systems worldwide
  - CRS-aware distance calculations for accurate spacing
  - Planting block grid layer (fishnet) aligned to row and column spacing
  - Population density statistics per hectare (trees/ha)
  - Multi-format export: Shapefile, GeoPackage, KML, and DXF
  - Background processing using QgsTask to keep the interface responsive
